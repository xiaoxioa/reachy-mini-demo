## 06-17:M1.5 多人注意力管理

### M1.5-a.5 — DOA 可信度信号 (`7bc20d1`)

- **DOA 常开固化**:说话时也算(XVF3800 AEC 能拾外部方向);只改算不改转
- **双窗并存**:
  - 老窗 `SND_WIN_S`(1.5s)→ `st.sound_resid`(SEEK/声源转向,不动)
  - 新窗 `DOA_WIN_S`(~2s)→ `st.doa_resid_stable` / `doa_confident` / `doa_at`
- **可信度 = 纯 IQR**:`confident = fresh & IQR < 25° & n ≥ 6`
  - 镜像翻转(两簇 ~97°)→ IQR≈90 自动 False
  - 可用区 IQR 3-15° → True
- **⚠️ 语义:confident = "稳"非"对"** — 稳定镜像错值仍 True,靠视觉层兜

### M1.5 ATTEND-01 — 四件一提交 (`72e86ca`)

#### a. 方向门控

- engaged 中 DOA 确信声源在 ±55° 范围外 → append 注入静音帧
- 不 confident / 不 fresh → 默认放行(宁可多听不可误杀)
- `--no-gate` 回退

#### b. 二次唤醒切换(核心难点)

- **触发**:engaged KWS 命中"小艺",排除 A 自喊(confident + |resid| ≤ 55°)
- **三档 DOA 方向**:
  1. confident + fresh → 直转 `A方向 + resid`(精确)
  2. fresh 不 confident → 粗方向 `A方向 + sign(resid) × 70°`
  3. 不 fresh → 反向 `离A中心方向 × 70°`(兜底)
- **turn 阶段全压锁(v2 修复)**:
  - v1 教训:turned_away = 35° 不够,A 在 ~60° FOV 内仍被提前锁住
  - v2:turn 阶段完全无视任何脸;到位后切 sweep 才放开(带 turned_away 门)
  - sweep 以 switch_target 为中心(非 A 位置,避免扫回 A)
- **失败兜底**:超时 8s 未锁 B → 回 A(RETURNING 重锁,不回 armed)
- **切换后**:close + open 新会话 + 招呼一句(复用 greet)
- **sticky_reset**:切换启动时清 FaceSelector 记忆,防粘住 A

#### c. sticky 多脸粘滞

- 新增 `FaceSelector` 类(`vision_worker.py`):
  - 跨帧匹配:欧距 ≤ 0.18(归一化坐标)
  - 切换压力:另一张脸连续 8 帧(~0.3s@27fps)大于当前 × 120% 才切
  - 消失 / 所有脸太远 → 退回 argmax
- `--no-sticky` 回退原始每帧 argmax

#### d. SEEK 两阶段

- **direct**:confident DOA → 直转到目标角度;途中压锁(|resid| < 12° 正前豁免)
- **nearby**:到位后 ±25° 慢扫 2.5s 找脸;锁到 → TRACK
- **full**:超时或不 confident → 原始 ±88° 全场扫(不变)

### DOA 分层防御体系(M1.5 确立的核心设计)

```
        DOA 层                    行为层                  视觉层
    ┌──────────┐            ┌──────────────┐        ┌──────────┐
    │ confident│──精确──→   │ 直转到角度   │──到位→ │ 附近找脸 │
    │ (IQR筛)  │──粗方向──→ │ sign×70°转   │──到位→ │ 附近找脸 │
    │ 不稳定   │──反向──→   │ 离A方向转    │──到位→ │ 附近找脸 │
    │ 镜像错   │            │ (视觉兜底)   │        │ 见脸即锁 │
    └──────────┘            └──────────────┘        └──────────┘
         ↑ 只筛抖动              ↑ 三档降级             ↑ 最终裁判
```
